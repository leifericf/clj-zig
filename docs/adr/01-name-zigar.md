# ADR 01: Name the experiment Zigar

Date: 2026-06-16

## Context

The experiment needed a working name before the dossier could refer to
it consistently.

## Decision

Use `Zigar` as the working name.

## Consequences

The name carries two jokes that set the tone: `Zigarette` (writing
low-level native Zig inside a high-level Lisp can feel a little dirty,
like smoking) and an inverted "close, but no cigar" (a narrowly
achieved success, not a near miss). It reads small and sharp, which
matches the proof-of-concept scope.

## Alternatives

`Zigarette` in full was considered; `Zigar` is shorter and cleaner
while keeping the same joke.
